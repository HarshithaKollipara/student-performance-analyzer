# Student Performance Analyzer

## Description
A Python-based application to manage and analyze student academic records.
The project ranks students, calculates class statistics, and supports searching
using fundamental data structures and algorithms.

## Features
- Add and display student records
- Calculate average, highest, and lowest marks
- Rank students using sorting algorithms
- Search students by name

## Technologies Used
- Python
- Data Structures (Lists, Dictionaries)
- Sorting and Searching
